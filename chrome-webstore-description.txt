ProxyFox - 智能代理管理专家

ProxyFox 是一款功能强大且易于使用的 Chrome 代理管理扩展，专为需要频繁切换网络环境的用户设计。无论您是开发人员、安全专家，还是需要访问不同地区内容的用户，ProxyFox 都能为您提供高效、安全的代理管理解决方案。

核心功能特色：
• 一键切换 - 在多个代理配置间快速切换，无需复杂设置
• 智能白名单 - 支持全局和代理专用白名单，*.domain.com 自动匹配根域名
• 多协议支持 - 完整支持 HTTP、HTTPS、SOCKS4、SOCKS5 代理协议
• 认证管理 - 安全存储代理用户名和密码，自动处理认证
• 配置导入导出 - 轻松备份和分享代理配置，支持批量管理
• 国际化界面 - 支持简体中文、繁体中文、英文、日文、韩文五种语言

安全与隐私：
• 本地存储 - 所有配置和凭据仅存储在本地，保护隐私安全
• 最小权限 - 仅请求必要的扩展权限，符合 Chrome MV3 安全标准
• 开源透明 - 代码开源，接受社区审查和贡献

智能特性：
• 自动域名展开 - 输入 *.example.com 自动包含 example.com 根域名
• 代理健康检测 - 实时监控代理连接状态
• 注释支持 - 白名单支持行内和整行注释，便于管理大型规则集
• 状态同步 - 自动验证和同步代理状态，确保配置正确生效

适用场景：
✓ 开发测试不同网络环境
✓ 安全研究和渗透测试  
✓ 访问地区限制内容
✓ 网络隐私保护
✓ 企业网络环境切换

🇺🇸 ENGLISH VERSION
========================================

ProxyFox - Smart Proxy Management Expert

ProxyFox is a powerful and user-friendly Chrome proxy management extension designed for users who need to frequently switch network environments. Whether you're a developer, security expert, or user needing access to geo-restricted content, ProxyFox provides an efficient and secure proxy management solution.

Core Features:
• One-Click Switching - Quickly switch between multiple proxy configurations without complex setup
• Smart Whitelist - Supports global and proxy-specific whitelists, *.domain.com automatically matches root domains
• Multi-Protocol Support - Full support for HTTP, HTTPS, SOCKS4, SOCKS5 proxy protocols
• Authentication Management - Securely store proxy credentials and handle authentication automatically
• Import/Export - Easy backup and sharing of proxy configurations with batch management support
• Internationalization - Supports 5 languages: Simplified Chinese, Traditional Chinese, English, Japanese, Korean

Security & Privacy:
• Local Storage - All configurations and credentials stored locally only, protecting privacy
• Minimal Permissions - Requests only necessary extension permissions, compliant with Chrome MV3 security standards
• Open Source - Transparent code open for community review and contribution

Smart Features:
• Auto Domain Expansion - Input *.example.com automatically includes example.com root domain
• Proxy Health Detection - Real-time monitoring of proxy connection status
• Comment Support - Whitelist supports inline and full-line comments for managing large rule sets
• State Synchronization - Automatic validation and synchronization of proxy states

Use Cases:
✓ Development testing in different network environments
✓ Security research and penetration testing
✓ Accessing geo-restricted content
✓ Network privacy protection
✓ Enterprise network environment switching
